# Tech Workshop Sign-up App

A modern, minimalist web application built with HTML, CSS, and JavaScript that allows users to sign up for a technology workshop. The project features a clean, open-theme UI designed for the best user experience.

## Features
* **Advanced Form Validation:** Validates required fields, email format, and age restrictions (18+) using sleek **SweetAlert2** pop-up notifications instead of standard browser alerts.
* **Dynamic Rendering:** Displays a real-time, responsive list of enrolled participants below the form.
* **Participant Management:** Users can easily **Edit** or **Delete** participants directly from the list, with UI elements powered by **Font Awesome** icons.
* **Search Functionality (Task 4 - Option C):** Includes a real-time search input to filter the participant list by full name or email address.
* **Persistent Storage (Task 4 - Option D):** All participant data is securely saved to the browser's `localStorage`, ensuring data persists even after a page refresh.

## Technologies Used
* **Semantic HTML5**
* **CSS3** (Flexbox, reusable classes, clean UI architecture)
* **Vanilla JavaScript** (ES6+, DOM Manipulation, Array methods, LocalStorage API)
* **[SweetAlert2](https://sweetalert2.github.io/)** (For modern feedback dialogs)
* **[Font Awesome](https://fontawesome.com/)** (For intuitive action icons)

## Setup & Run
1. Clone the repository or extract the ZIP archive.
2. Ensure an active internet connection to load the SweetAlert2 and Font Awesome CDN links.
3. Open `index.html` in any modern web browser to start the application. No local server or build tools are required.
